Help / Glossary
Batch Size - How many CPEs are grouped per cycle before writing results.
Timeout (sec) - Max seconds to wait for a device response before moving on.
Threads - Parallel workers running checks at the same time.
Rules Source - Where DNS rules come from (db, static, or mixed).
Mode - audit = check only, fix = apply fixes, fix+reboot = fix then reboot.
Progress Interval - How often the worker writes progress updates.
